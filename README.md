# StoreFlow

A Next.js implementation of StoreFlow, an e-commerce experience with gifting and packaging options plus admin/staff operations views.

## Features included

- Customer storefront with search and category filtering.
- Cart management with quantity updates.
- Gift workflow (recipient, note fee, packaging pricing).
- Dynamic checkout pricing (items + gift options + shipping).
- Admin dashboard for order status, gift details, inventory, packaging, and a sales snapshot.
- Staff dashboard for assigned orders and delivery status updates.

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
